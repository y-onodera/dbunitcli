DBUnit CLI ヘルプ

コマンド

  • Convert
    データ形式を変換する
  • Compare
    データセットを比較して差分を検出する
  • Generate
    テンプレートからファイルを生成する
  • Run
    SQL やスクリプトを実行する
  • Parameterize
    パラメータでバッチ処理する

リファレンス

  • Dataset Load Form
    データセット入力フォームの構成
  • Dataset Settings
    Dataset Settings ファイルの書式
  • Xlsx Schema
    Xlsx 入力のマッピング定義
