Run

SQL やスクリプトを実行します。データソースに対して SQL 文・ANT スクリプト・バッチファイルを実行します。

オプション

オプション説明
scriptType実行するスクリプトの種類。sql / ant / bat / cmd から選択（デフォルト: sql）
srcData実行対象ファイルの指定。拡張子は scriptType に応じて自動設定（sql→.sql, ant→.xml, bat→.bat, cmd→.cmd）
templateOptionテンプレートエンジンのオプション（sql のみ有効）
jdbcOptionJDBC 接続設定（sql のみ有効）
baseDirスクリプトの作業ディレクトリ（ant / bat / cmd のみ有効）
antTarget実行する ANT ターゲット名（ant のみ有効）

srcData の入力フォーム詳細は Dataset Load Form を参照してください。

SQL 実行例

SQL ファイルをテンプレートエンジンで加工してからデータベースに対して実行します。パラメータは srcData で渡したデータセットの各行がテンプレート変数として展開されます。

設定項目値の例
scriptTypesql
srcData.srcparams/run_params.csv（実行パラメータを格納した CSV）
templateOption.templateRenderst4（StringTemplate 4 を使用）
jdbcOption.urljdbc:h2:./db/testdb
jdbcOption.usersa

SQL ファイル内では <param.COLUMN_NAME> の形式でパラメータを参照できます。srcData の行数分だけ SQL が繰り返し実行されます。

ANT 実行例

Apache ANT の build.xml を実行します。srcData で指定したファイル一覧（.xml）が対象となり、各ファイルに対して antTarget で指定したターゲットが実行されます。パラメータは ANT プロパティとして渡されます。

設定項目値の例
scriptTypeant
srcData.srcscripts/（build.xml を格納したディレクトリ）
baseDirwork/（ANT 実行時の作業ディレクトリ）
antTargetdeploy（実行するターゲット名。省略時はデフォルトターゲット）

srcData で渡したデータセットの各行は ANT プロパティ（property name="COLUMN_NAME" value="..."）として build.xml 内から参照できます。

BAT / CMD 実行例

Windows バッチファイル（.bat / .cmd）を実行します。srcData で指定したファイル一覧が対象となり、各ファイルを baseDir を作業ディレクトリとして順に実行します。パラメータは環境変数として渡されます。

設定項目値の例
scriptTypebat
srcData.srcscripts/（.bat ファイルを格納したディレクトリ）
baseDirwork/（バッチ実行時の作業ディレクトリ）

srcData で渡したデータセットの各行は環境変数としてバッチファイル内から %COLUMN_NAME% の形式で参照できます。cmd を選択した場合も動作は同様です。
