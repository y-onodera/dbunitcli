Parameterize

パラメータでバッチ処理します。パラメータデータセットの各行を入力として、対象コマンドを繰り返し実行します。

オプション

オプション説明
unitテンプレートへのパラメータ受け渡し単位（行・テーブル・データセット）
parameterizecmdParam をパラメータで展開するテンプレートとして扱うかどうか
ignoreFailcompare で想定外の差分が出た場合も他のコマンドを続行する
cmdデータ駆動で実行する対象コマンド
cmdParamコマンド用パラメータファイル。ファイル名はパラメータデータセットから動的に決定
template既定のテンプレートファイル（cmdParam がある場合は無視）
paramDataパラメータデータセットのソース
templateOptionテンプレートエンジンのオプション

paramData の入力フォーム詳細は Dataset Load Form を参照してください。

unit の動作

unit はテンプレートへのパラメータ受け渡し単位を指定します。

値説明
rowparamData の行ごとに 1 回コマンドを実行
tableparamData のテーブルごとに 1 回コマンドを実行
datasetparamData 全体を 1 つとして 1 回コマンドを実行

具体例

Compare をパラメータ駆動で繰り返す

比較対象のディレクトリパスをパラメータ CSV で管理し、複数の比較を一括実行する例です。

paramData に読み込む CSV（例: params/compare_params.csv）:

OLD_DIR,NEW_DIR,SETTING
data/old/orders,data/new/orders,settings/orders.json
data/old/products,data/new/products,settings/products.json

template に指定する Compare パラメータファイルのテンプレート（例: templates/compare_param.st4）:

{
  "oldData": { "src": "<param.OLD_DIR>" },
  "newData": { "src": "<param.NEW_DIR>" },
  "setting": "<param.SETTING>"
}
設定項目値の例
unitrow
parameterizefalse
ignoreFailtrue（差分があっても次の行を処理する）
cmdcompare
templatetemplates/compare_param.st4
paramData.srcparams/compare_params.csv

paramData の各行（OLD_DIR・NEW_DIR・SETTING）が <param.列名> としてテンプレートに展開され、行数分だけ Compare が実行されます。

cmdParam でパラメータファイルを動的に切り替える

パラメータ CSV の列値でコマンドパラメータファイルのパスを動的に決定する例です。parameterize を有効にすると cmdParam の文字列もテンプレートとして展開されます。

paramData に読み込む CSV（例: params/targets.csv）:

TARGET
orders
products
customers
設定項目値の例
unitrow
parameterizetrue
cmdconvert
cmdParamparams/<param.TARGET>.json
paramData.srcparams/targets.csv

1 行目では cmdParam が params/orders.json に展開され Convert が実行されます。2 行目では params/products.json、3 行目では params/customers.json が使われます。

Generate をパラメータ駆動で繰り返す

テーブル名の一覧から SQL ファイルを一括生成する例です。

paramData に読み込む CSV（例: params/tables.csv）:

SCHEMA,TABLE_NAME
public,orders
public,products
設定項目値の例
unitrow
parameterizefalse
cmdgenerate
templatetemplates/generate_param.st4
paramData.srcparams/tables.csv

各行の SCHEMA・TABLE_NAME がテンプレートに渡され、テーブルごとに Generate が実行されます。
