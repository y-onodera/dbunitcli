Parameterize

Batch process with parameters. Executes a target command repeatedly, driven by rows from a parameter dataset.

Options

OptionDescription
unitPass parameter to template by row, table, or dataset
parameterizeWhether cmdParam is a template populated by parameters
ignoreFailContinue other commands when compare finds unexpected differences
cmdData-driven target command
cmdParamParameter file for command; dynamically set filename from parameter dataset
templateDefault template file (ignored when cmdParam exists)
paramDataParameter dataset source
templateOptionTemplate engine options
