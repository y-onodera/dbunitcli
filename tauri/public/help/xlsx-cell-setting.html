Xlsx Cell Setting

Xlsx シート上の任意のセル位置から値を拾い集めてテーブルを構成するための設定を編集するダイアログです。帳票・フォーマット化された Xlsx など、行方向に整列していないデータの抽出に利用します。全体構成は Xlsx Schema を参照してください。

設定項目

フィールド説明
sheetName抽出対象の Xlsx シート名。データセットがロード済みの場合、候補がデータリストから選択できます。
tableName抽出結果に付与する論理テーブル名。
headerカラム名の並び。カンマ区切りで指定します。各レコードのセルアドレスと同じ順序で対応させます。
rows1 レコード分のセルアドレスをカンマ区切りで指定します。行を追加することで複数レコードを定義できます。
addFileInfo元ファイル名などのファイル情報カラムを結果に追加するかどうか。

記述例

帳票の見出しセルから 1 レコードを抽出

フィールド値
sheetNameInvoice
tableNameinvoice_header
headerinvoice_no,issue_date,customer_name,total
rowsB2,B3,B4,F10

複数の帳票が 1 シートに並ぶ場合は rows に行を追加し、それぞれの帳票のセルアドレスをカンマ区切りで指定します。

注意事項

  1. sheetName と tableName の両方が入力されていない行は保存対象から除外されます。
  2. header と各 rows のセルアドレス数は一致させてください。
  3. セルアドレスは Xlsx の A1 形式で指定します。
