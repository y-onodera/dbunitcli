Generate

テンプレートまたは組み込み定義からファイルを生成します。ソースデータをもとに SQL・テキスト・Excel・固定長定義ファイルなどを出力します。

generateType

生成の種類を選択します。選択した種類に応じて、表示されるオプションが変わります。

値説明unit
txtStringTemplate 4 を使ったテキストファイル生成（SQL・CSV・任意テキスト等）record / table / dataset
xlsxjxls を使った .xlsx テンプレートへのデータ埋め込みrecord / table / dataset
xlsjxls を使った .xls テンプレートへのデータ埋め込みrecord / table / dataset
settingsDataset Settings ファイルのひな型を生成dataset（固定）
sqlINSERT / UPDATE / DELETE / CLEAN_INSERT SQL ファイルを生成table（固定）
xlsxTemplatejxls テンプレートのひな型 .xlsx を生成dataset（固定）
fixedColumnDef固定長出力用のカラム幅定義ファイルを生成table（固定）

unit

生成の単位を指定します。デフォルトは record です。
settings / xlsxTemplate は dataset、sql / fixedColumnDef は table に固定されます。

値説明
record行ごとに 1 ファイルを出力
tableテーブルごとに 1 ファイルを出力
datasetデータセット全体から 1 ファイルを出力

共通オプション

項目説明
templateテンプレートファイルのパス（txt / xlsx / xls タイプで使用）
result生成結果ファイルを配置するディレクトリ
resultPathresult ディレクトリからの相対パス（テンプレート変数を使用可能）
outputEncoding出力ファイルの文字コード（txt タイプのみ、デフォルト: UTF-8）

txt

StringTemplate 4 テンプレートファイルを使い、テキストファイルを生成します。template オプションでテンプレートファイルを指定します。テンプレートエンジンの詳細設定は template option の展開ボタンから設定します（Template 参照）。

テンプレート変数

unit の選択によってテンプレート内で使用できる変数が異なります。

変数名説明recordtabledataset
rowNumber行 / テーブルのインデックス（0 始まり、dataset は常に 0）○○○
tableNameテーブル名○○—
columnsカラムの配列（各要素に columnName・dataType）○○—
primaryKeys主キーカラムの配列（各要素に columnName・dataType）○○—
columnsExcludeKey主キー以外のカラムの配列（各要素に columnName・dataType）○○—
row現在行のデータ（列名 → 値 の Map）○——
rowsテーブル全行のリスト（各要素は列名 → 値 の Map）—○—
dataSetテーブル名をキーとする Map。各値は table スコープと同じ構造——○

xlsx / xls

jxls テンプレートファイルを使い、Excel ファイルを生成します。template オプションでテンプレートファイルを指定します。jxls 固有の設定は jxls option の展開ボタンから設定します（Jxls 参照）。

settings

データセットのカラム情報から Dataset Settings ファイルのひな型を生成します。unit は dataset に固定です。template は使用しません。

sql

テーブルのカラム情報から SQL ファイルを自動生成します。unit は table に固定です。template は使用しません。

xlsxTemplate

jxls テンプレートのひな型となる .xlsx ファイルを生成します。unit は dataset に固定です。template は使用しません。

fixedColumnDef

固定長出力（Result Form — fixed）で使用するカラム幅定義ファイルを生成します。unit は table に固定です。以下の項目を設定します。

項目説明
fixedLength各カラムの幅をカンマ区切りで指定（例: 4,10,20）。カラム数より少ない場合、残りのカラムには defaultLength が使われる
defaultLengthfixedLength で幅が指定されなかったカラムに適用するデフォルト幅（数値）
alignパディングの方向: left（右余白）/ right（左余白）
