Generate

テンプレートからファイルを生成します。ソースデータとテンプレートエンジンを用いて、SQL や CSV などの出力ファイルを作成します。

generateType

値説明
txtStringTemplate 4 を使ったテキストファイル生成（SQL・CSV 等）。Template オプションが適用されます
xlsxjxls を使った xlsx テンプレートへのデータ埋め込み。Jxls オプションが適用されます
xlsjxls を使った xls テンプレートへのデータ埋め込み。Jxls オプションが適用されます
settingsデータセット設定ファイルのひな型を生成
sqlINSERT / UPDATE / DELETE / CLEAN_INSERT SQL ファイルを生成
xlsxTemplatejxls テンプレートのひな型 xlsx を生成

unit

生成の単位を指定します。デフォルトは record です。
ただし generateType が settings / xlsxTemplate の場合は dataset、sql の場合は table に固定されます。

値説明
record行ごとに 1 ファイルを出力
tableテーブルごとに 1 ファイルを出力
datasetデータセット全体から 1 ファイルを出力

その他のオプション

オプション説明
templateテンプレートファイル
result生成結果ファイルを配置するディレクトリ
resultPathresult ディレクトリからの相対パス
outputEncoding出力ファイルのエンコーディング（txt タイプのみ）
