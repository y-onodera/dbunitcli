Result Form

各コマンドで「出力先と出力形式」を指定するための共通フォームです。Convert の convertResult、Compare の出力設定などで使用されます。

フォームの構成は Result Type 選択により変わります。選択した形式ごとに表示される項目が異なります。

Result Type 選択

出力データの形式を選択します。以下の 6 種類から選択できます。

形式説明
csvCSV（区切り文字付きテキスト）で出力。拡張子・文字コードを変更可能
xlsExcel 97-2003 形式（.xls）で出力
xlsxExcel 2007 以降の形式（.xlsx）で出力
tableJDBC 接続先の DB テーブルへ書き込む
formatString.format() 書式でテキストとして出力
fixed固定長テキストで出力

共通項目（table 以外）

Result Type が table 以外の場合に表示される共通項目です。

項目説明
result出力先ディレクトリのパス
resultPath出力先パスのテンプレート文字列。テーブル名などを変数として使用可能
exportEmptyTableON にすると、データ行が 0 件のテーブルも出力する（デフォルト: ON）
exportHeaderON にすると、出力ファイルにヘッダ行を含める（デフォルト: ON）

csv

テーブルを CSV ファイルとして出力します。1 テーブルにつき 1 ファイルが生成されます。

項目説明
outputEncoding出力ファイルの文字コード（デフォルト: UTF-8）
outputExtension出力ファイルの拡張子（デフォルト: .csv）。TSV として出力したい場合は .tsv を指定

出力例:

テーブル名: users
result    = output/
outputEncoding = UTF-8

生成ファイル: output/users.csv
id,name,age
1,Alice,30
2,Bob,25

xls

テーブルを Excel 97-2003 形式（.xls）で出力します。excelTable の設定により、テーブルをどのシートに書き込むかを制御します。

項目説明
excelTableExcel 出力時のシート配置モード（デフォルト: SHEET）。SHEET: 1 テーブル = 1 シート、BOOK: 全テーブルを 1 ブックにまとめる

xlsx

テーブルを Excel 2007 以降の形式（.xlsx）で出力します。項目は xls と同じです。

table

データを JDBC 接続先の DB テーブルへ書き込みます。result / resultPath / exportEmptyTable / exportHeader は表示されず、代わりに以下の項目が表示されます。

項目説明
opDB への書き込み操作（下表参照）
jdbc接続先の JDBC 接続情報（URL・ユーザ・パスワードなど）

op（DB 操作）

値説明
CLEAN_INSERT既存データを全削除してから INSERT（デフォルト）
INSERT既存データを残したまま INSERT
UPDATE主キーで一致する行を UPDATE
REFRESH一致行は UPDATE、存在しない行は INSERT
DELETE主キーで一致する行を DELETE

format

java.lang.String.format() の書式文字列を使い、各行のデータをテキスト出力します。

項目説明
outputEncoding出力ファイルの文字コード（デフォルト: UTF-8）
formatString.format() 書式文字列。例: %s,%s\n
outputExtension出力ファイルの拡張子（デフォルト: .txt）

出力例:

テーブル:
  id=1, name=Alice
  id=2, name=Bob

format = %s:%s\n

生成ファイル:
1:Alice
2:Bob

fixed

各カラムを固定長バイト幅でパディングしたテキストファイルとして出力します。

項目説明
outputEncoding出力ファイルの文字コード（デフォルト: UTF-8）
outputExtension出力ファイルの拡張子（デフォルト: .txt）
fixedColumnDef各カラムの固定長幅定義ファイルのパス。カラムごとのバイト幅をカンマ区切りで記述したファイルを指定
fixedLengthTypeバイト幅の計算方法: char（文字数基準、デフォルト）/ byte（バイト数基準）

出力例:

テーブル:
  id=1, name=Alice

fixedColumnDef: 4,10
fixedLengthType: char

生成ファイル:
1   Alice
