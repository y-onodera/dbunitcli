Template

テンプレート処理のための設定です。テンプレートエンジンは StringTemplate 4 (ST4)。式は既定で $...$ の記法を用い、入力データセットの各行をパラメータとして埋め込みます。以下の場面で利用されます。

  • Generate — 入力データセットをもとにファイル（SQL / テキスト / xlsx / 設定ファイル等）を生成
  • Run — SQL / ANT スクリプトを実行する前にテンプレートとしてレンダリング
  • Parameterize — パラメータごとにコマンド引数や設定をレンダリング
  • Dataset Load Form の templateGroup — 入力読み込み時にテンプレートを適用（任意）

設定項目

フォームの template option 展開ボタンから設定します。

項目説明既定
encodingテンプレートファイルの文字コード。Generate の template ファイル等に適用システム既定の文字コード
templateGroupStringTemplate グループファイル（.stg）のパス。共通マクロ（サブテンプレート）を別ファイルにまとめて読み込みます。ワークスペースの template/ ディレクトリを基準に解決空（グループなし）
templateParameterAttributeテンプレート内で入力行にアクセスする属性名。例えば param を指定すると、テンプレートに渡されたレコードは $param.columnName$ で参照できます。空文字を指定した場合は属性でラップせず、カラム名で直接参照しますparam
templateVarStartテンプレート式の開始文字（1 文字）$
templateVarStopテンプレート式の終了文字（1 文字）$

xlsx 出力時の追加オプション

Generate で generateType = xlsx を選んだ場合に適用される数式処理系のオプションです（内部的に Apache POI で処理）。

項目説明既定
formulaProcessxlsx 出力時に数式セルを処理するか。false にすると低メモリで動作しますが、数式内のセル参照が行追加にあわせて更新されませんtrue
evaluateFormulasformulaProcess が有効なときに Excel 数式を評価するかtrue
forceFormulaRecalc出力ファイルを Excel で開いたときに自動再計算させるフラグを立てるfalse
fastFormulaProcess高速な数式プロセッサを使用するfalse
deleteBlankCells数式評価後に空セルを削除するfalse

パラメータのバインド

入力データセットの各行は、テンプレートに対して templateParameterAttribute で指定した属性名のオブジェクトとして渡されます。

例 1: templateParameterAttribute = param（既定）

入力行 (1 件):
  tableName = USERS
  columnNames = id, name
  values = 1, 'Alice'

テンプレート (insert.sql):
  INSERT INTO $param.tableName$ ($param.columnNames$) VALUES ($param.values$);

レンダリング結果:
  INSERT INTO USERS (id, name) VALUES (1, 'Alice');

例 2: templateParameterAttribute を空に指定（直接バインド）

テンプレート (insert.sql):
  INSERT INTO $tableName$ ($columnNames$) VALUES ($values$);

レンダリング結果:
  INSERT INTO USERS (id, name) VALUES (1, 'Alice');

例 3: templateVarStart / templateVarStop を変更

テンプレートに $ を多く含む SQL（PostgreSQL の $$ 区切り等）を扱うときに、式の区切り文字を別の文字に変更できます。

指定:
  templateVarStart = <
  templateVarStop  = >

テンプレート:
  CREATE FUNCTION f() RETURNS void AS $$ ... $$ LANGUAGE sql;
  -- <param.tableName> をここで使える

結果:
  区切りが <...> に切り替わるため、$$ はそのまま出力される

templateGroup（共通マクロ）

StringTemplate のグループファイル（.stg）に再利用可能なサブテンプレートを定義し、本体テンプレートから呼び出せます。

グループファイル (macros.stg):
  quote(s) ::= <<'<s>'>>

本体テンプレート (insert.sql):
  INSERT INTO $param.tableName$ VALUES ($param.name:quote()$);

レンダリング結果:
  INSERT INTO USERS VALUES ('Alice');
