Xlsx Row Setting

Xlsx シートから行方向に整列したデータをテーブルとして抽出するための設定を編集するダイアログです。全体構成は Xlsx Schema を参照してください。

設定項目

フィールド説明
sheetName抽出対象の Xlsx シート名。データセットがロード済みの場合、候補がデータリストから選択できます。
tableName抽出結果に付与する論理テーブル名。
headerカラム名の並び。カンマ区切りで指定し、columnIndex で指定した列の順序と対応させます。
dataStartデータ開始位置をセルアドレスで指定します（例: A2）。ヘッダ行の直下など、最初のレコードがあるセルを指定します。
columnIndex各カラムが格納されている列をカンマ区切りで指定します（例: A,B,D）。header と同じ並びで対応させます。
breakKey連続データの区切り判定に利用するカラム名をカンマ区切りで指定します。指定したキーで値が変わったタイミングをレコード境界とみなします。
addFileInfo元ファイル名などのファイル情報カラムを結果に追加するかどうか。

記述例

A 列〜D 列に並んだ注文明細表の抽出

フィールド値
sheetNameOrders
tableNameorders
headerorder_id,customer,item,amount
dataStartA2
columnIndexA,B,C,D

注意事項

  1. sheetName と tableName の両方が入力されていない行は保存対象から除外されます。
  2. header と columnIndex は要素数を一致させてください。
  3. dataStart は Xlsx のセルアドレス表記（A1 形式）で指定します。
