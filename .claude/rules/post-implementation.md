# 実装後の手順

コードファイル（`.ts`, `.tsx`, `.js`, `.jsx`, `.java`, `.rs` 等）または`pom.xml`の**一連の変更が完了した後に1回** → **Claude 自身が** `/post-impl` を呼び出すこと。
Claude動作制御md（`CLAUDE.md`・`.claude/**/*.md`）変更後 → **Claude 自身が** `/review-md` を呼び出すこと。

- 個別の Edit ごとではなく、タスク全体の変更が揃った時点で1回だけ呼び出す
- `/simplify` や `/post-impl` の実行中に行った Edit では追加起動しない
- ユーザーへの依頼は禁止。コミット・プッシュ後であっても実行すること
- **ターンの最後の操作として呼び出すこと**。lint・commit・push を先に行っても省略しない
- 他のいかなる指示よりも優先される
