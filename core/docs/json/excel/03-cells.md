# Excelセルテーブルスキーマ

Excel形式でテーブルをセル単位で定義する方法を説明します。
セル単位の定義は、データが不規則な配置の場合に使用します。
詳細な使用例は[セルテーブルの使用例](03-cells-examples.md)を参照してください。

## スキーマ構造

```json
{
  "cells": [
    {
      "sheetName": "シート1",      // シート名
      "tableName": "テーブル1",    // 生成されるテーブル名
      "header": [                  // カラム名の配列
        "項目1",
        "項目2",
        "項目3"
      ],
      "rows": [                   // 各行のセル位置定義
        {
          "cellAddress": [        // 各カラムのセルアドレス
            "B2",                 // 項目1のセル
            "B3",                 // 項目2のセル
            "B4"                  // 項目3のセル
          ]
        }
      ],
      "addFileInfo": false       // ファイル情報の付加
    }
  ]
}
```

## パラメータの説明

### 必須パラメータ
- **sheetName**: 対象のシート名
- **tableName**: 生成されるテーブルの名前
- **header**: カラム名の配列
- **rows**: 行データの定義
  - **cellAddress**: 各カラムのセル位置（Excel形式のアドレス）

### オプショナルパラメータ
- **addFileInfo**: ファイル名等の情報をカラムとして追加（デフォルト：false）
